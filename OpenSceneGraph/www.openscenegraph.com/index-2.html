The OpenSceneGraph Project Website

Details
Category: OpenSceneGraph

The OpenSceneGraph is an open source high performance 3D graphics toolkit, used by application developers in fields such as visual simulation, games, virtual reality, scientific visualization and modelling. Written entirely in Standard C++ and OpenGL it runs on all Windows platforms, OSX, GNU/Linux, IRIX, Solaris, HP-Ux, AIX and FreeBSD operating systems. The OpenSceneGraph is now well established as the world leading scene graph technology, used widely in the vis-sim, space, scientific, oil-gas, games and virtual reality industries.

VulkanSceneGraph Project

Details
Category: News

The VulkanSceneGraph project is now underway, combining Vulkan and C++17 to create a next-gen scene graph.  Developed under the auspices of OpenSceneGraph Project Lead, the new project is the natural successor to the OpenSceneGraph, with the OpenSceneGraph project remaining supported and moved to a maintenance phase.  The new project is hosted on github and will be developed over the coming year with the goal of a stable 1.0 release late 2019.  For support head over to the vsg-users googlegroup.
