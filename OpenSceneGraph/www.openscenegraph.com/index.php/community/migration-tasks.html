This is the migration state of the openscenegraph project site. If you are willing to collaborate you can ask for author rights to the site admins.

Once you are logged in follow the next steps:

  • Edit this article and put your name in the column Asigned to.
  • Change State to : In progress.
  • Put the new link into the column Openscenegraph.com
  • Once you have migrated the content set the status to awaiting review
  • If all looks good the publisher should set the status to complete.
