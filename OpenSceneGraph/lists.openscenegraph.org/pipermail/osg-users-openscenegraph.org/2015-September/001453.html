cbuchner1 wrote:
> 
> did I understand correctly that the time warping feature cannot be disabled, as it is a built-in feature of the Oculus SDK?
> 


My understanding is that time warp is enabled by default. And I have not found a simple way of disabling it. I have posed a question on the Oculus forum, but have not received any replies, yet.

But when I have read some more about how positional time warp works I discovered that to be able to use it the compositing layer must of type ovrLayerType_EyeFovDepth. And currently the osgoculusviewer is using the simpler ovrLayerType_EyeFov layer type, which only supports rotational time warp.
