Hi Trajce,


The osgmygui example is compiled against MyGUI 3.01 and will look for resource.xml file in current working directory to locate MyGUI media files. The template of resource.xml can be found at <MyGuiRepo>/CMake/Templates, which should also be installed to <InstallPrefix>/bin automatically.
