Hi,

For some of the standard dependencies I have made CMake scripts that helps building them from source. As of now the scripts supports:

* zlib
* libpng
* libjpeg
* libtiff
* FreeType
* GLUT
* GIFLIB
* MINIZIP

And have been tested with the most recent versions of Visual Studio as well as MingGW+GCC.

The build scripts are available here:
https://github.com/bjornblissing/osg-3rdparty-cmake
