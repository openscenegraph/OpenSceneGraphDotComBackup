Hi all,

For the github+binary combination, there is a dedicated doc page on github:
https://help.github.com/articles/distributing-large-binaries/

In short, it is possible to attach binaries to each of the releases. There
is no restriction on the total size of the binary files, however, each
individual file cannot exceed more than 2 Gb.
