1.) Make sure you have set 


Code:
geometry.setDataVariance(osg::Object::DYNAMIC)



2.) To clear the "cached" data if you use display lists:


Code:
geometry.dirtyDisplayList()



or if you use vertex buffer objects:


Code:
vertices.dirty()
