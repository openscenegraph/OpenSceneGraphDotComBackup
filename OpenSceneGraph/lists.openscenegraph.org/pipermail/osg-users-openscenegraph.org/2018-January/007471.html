The error looks like the CMake has told the OSG that is has found
freetype but for some reason the libs is has directed the OSG to use
for freetype aren't available for some reason.  Something odds has
happen here as I and others have been using Ubuntu+Kubunut including
12.04 without issues with freetype.  How did you go about installing
freetype and it's dev headers?

To see what the FindFreetype.cmake has picked up for freetype have a
look at the freetype entries it has found by using ccmake . on the
command line then looking through to the FREETYPE_* entries.
