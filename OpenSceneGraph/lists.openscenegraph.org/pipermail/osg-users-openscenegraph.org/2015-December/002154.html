Hello Robert and Emmanuel!

I tested this on Windows (3.4.0 OSG, Qt 5.5.1, MSVC2013 32bit).
Screenshot attached in the attachment.

I used insulator dad-0.3.0-windows-vc12x86-testing.exe from my project:

http://dad.podsvirov.pro

What I did:

a) Set the module OpenSceneGraph
b) Launched a console in the folder "bin"
c) Set the environment variable OSG_FILE_PATH (you should be in osg-data)
d) Run "../share/OpenSceneGraph/bin/osgviewerQt.exe

At startup, I got two white Windows, but after small changes in the window using
of the mouse pointer is displayed normally.

Recently I conducted an experiment with OSG 3.5.1 and it worked immediately.

(Soon will upload OSG 3.5.1 installer dad-0.3.0-windows-vc12x64-unstable.exe)

I hope my comments are useful.

Additionally I will say that as part of their project development osgQtQuick (https://github.com/podsvirov/osgqtquick)
OSG and Ot Quick (Qt5) works fine.
