Sorry to hear Qt/OSG experience has been a bit rough.  Qt has been a bit of
moving target and generally awkward to work with as it does some odd things
with event handling, threading and state.  The exact interaction of all of
this has changed over time with newer versions of Qt breaking how things
used to work, and then fixes to get things working again break how it works
with older versions.

Personally I am not a Qt user so don't have the expertise for tracking
these changes and recommending fixes, it is possible to get it working as
there are plenty of OSG/Qt users out there.  I'm guessing there are mostly
just getting on with their work and haven't spotted your request for
assistance, or perhaps they don't know enough about the specifics of the
combination of OSG/Qt you are using to comment.
