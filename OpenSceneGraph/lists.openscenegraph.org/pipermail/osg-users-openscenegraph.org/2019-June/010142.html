When the widget is popped out, it causes the creation of a new OpenGL context. I see that you've told Qt to share things between the old and new context, which means you'll still be able to use some things in the new one, but I'd look into what can and can't be shared as it isn't everything. I'd guess that something that the text node is using can't be shared.
