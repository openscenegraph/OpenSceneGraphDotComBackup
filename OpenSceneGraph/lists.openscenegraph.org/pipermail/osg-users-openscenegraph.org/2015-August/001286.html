HI Erik,

The constructor takes a reference not a pointer so try:

 osg::ref_ptr < osg::Vec3Array > pCopy = new osg::Vec3Array(*pVertexArray,
osg::CopyOp::DEEP_COPY_ALL);

Noe the *.


Or use osg::clone() i.e.

  pCopy = osg::clone(pVertexArray->get());

Robert.
