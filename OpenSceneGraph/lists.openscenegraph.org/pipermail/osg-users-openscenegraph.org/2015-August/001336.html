On 26 August 2015 at 14:56, Julie Green <laroux92 at mail.ru> wrote:

> Oh, I forgot to set new vertex array.


I would recommend reusing vertex arrays rather than creating new ones each
frame.

You will also need to call geometry->dirtyBound() to make sure the bounding
volumes get updated to each frame so that as the geometry changes the view
frustum culling still can function correctly for the new geometry
positions.  Finally it'll be best to disable display lists for the geometry
so that original geometry isn't compiled into a list and used thereafter,
simply do geometry->setUseDisplayList(false) during the set up of the
geometry.
