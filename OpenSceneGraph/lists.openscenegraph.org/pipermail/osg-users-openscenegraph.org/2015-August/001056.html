> I believe we must be pretty close to stable 3.4.0 release as I had 
> only one submission to merge since rc8 that I tagged at the end of 
> last week!  The revision was to make osgViewer's X11 PixelBuffer query 
> for the SGIX extensions at runtime rather than use them directly at 
> compile time.  This change is intended to make it possible to run the 
> OSG on X11 systems that don't provide support for the SGIX extensions.
