Hello dear OSG-community,

The OSG 3.6.4 Windows binaries at https://objexx.com/OpenSceneGraph.html were refreshed with the latest 3rd party packages and the just-released Qt 5.13.2.

This refresh adds support for both the old osgQt (that is now in their Qt4 branch) and the new osgQOpenGL approach (from the current osgQt git master branch). This should let projects transition to osQOpenGL when they are ready (and can get it to work in their applications, which we haven't managed yet!).

Best regards,
