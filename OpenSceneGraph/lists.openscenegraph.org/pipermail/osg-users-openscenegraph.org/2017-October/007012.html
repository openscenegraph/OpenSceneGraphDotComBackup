Hi,

For the third party libraries I used Appveyor, which was pretty simple to setup. It has the ability to push built artifacts as to Github as releases. It is free for open source libraries, BUT I think their build time limit may impose problems. Their maximum allowed build time is 60 minutes and the VMs they are using are not super fast. I guess it would be hard to get OpenSceneGraph to finish a full build under 60 minutes.
