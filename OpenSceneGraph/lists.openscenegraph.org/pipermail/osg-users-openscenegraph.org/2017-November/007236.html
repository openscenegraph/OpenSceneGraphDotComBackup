Hi
You cannot use osg zip plugin to read arbitrary zip file...
Nevertheless osgDB serializer have several interfaces (Image,Object,Node,Archive,HeightField) that can be implemented in a plugin to read/write these osg types.
So one could create its own class heriting one of the interfaces and/or develop its own plugin in order to read from a zip.

Hope it helps
