I don't think it has been updated in years. Cal3D itself hasn't been
updated in a long time neither.  Someone has imported a version of the
osgCal code here (despite the SourceForge link, this is not the GPLed
fork below):

https://github.com/hlrs-vis/osgcal 


There is also the osgCal2 GPL-ed fork on SourceForge, but that hasn't
been updated since 2005 or so neither. 


I think the project is fairly dead, you may want to consider porting
your code to osgAnimation instead.
