Hi Sebastian,

Geode {...
            Geometry { ...
                             StateSet { ...
                                           GL_CULL_FACE ON
                                            Material { .... }
                                            CullFace {
                                                mode BACK
                                            }

For no culling use  GL_CULL_FACE OVERRIDE|OFF
To cull front faces use mode FRONT.
To cull all faces use mode FRONT_AND_BACK
Regards, Laurens.
