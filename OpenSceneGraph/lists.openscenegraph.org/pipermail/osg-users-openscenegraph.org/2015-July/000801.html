Michael,
it is lightweight image generator mainly for flight simulation although the
latest version has bullet vehicle plugin for ground sim as well. So it is
about moving models around in virtual space, mathematically controling
parts of the model (the model from the web has about 100 independently
controlable parts, so if one has heavy flight math model a very detailed
simulation can be done), Vegetation on the GPU, Particle effects, Shadow
Maps, Weather and Atmosphere model (commercial and a simple free one),
support for animation (experimental value driven from the OpenIG core and
the FBX) , integration with osgEarth (with atmosphere and ocean model), and
few more. All these are plugins. We are investigating Forward+ and Deffered
Rendering and Lighting at present so this might come in one of the next
release, with time.  A bit of history: The development was driven by few
projects that are flight sim related and it is already proved commercialy.
Good is that my client Compro decided to opensource it, for users in need
of lightweight image generator, again for flight sim but not only. Also,
since it is plugin based one can configure OpenIG with better looking
commercial plugins for more serious simulation, or use the free provided
for simplier cases.

There are some videos on YouTube (available on the bottom of the main web
site). It might gives you better idea.

Thanks for asking !

Nick
