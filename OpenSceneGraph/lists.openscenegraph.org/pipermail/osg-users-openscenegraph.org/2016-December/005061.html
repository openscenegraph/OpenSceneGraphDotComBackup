Not sure about PBO but FBO support in OSG works with multisampling.
See

Camera::attach(
  BufferComponent buffer,
  osg::Texture* texture,
  unsigned int level,
  unsigned int face,
  bool mipMapGeneration,
  unsigned int multisampleSamples,
  unsigned int multisampleColorSamples)

method.
