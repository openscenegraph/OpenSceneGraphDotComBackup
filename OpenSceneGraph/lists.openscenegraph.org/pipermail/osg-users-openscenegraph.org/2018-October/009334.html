Hello dear OSG-community,

OSG 3.6.3 Windows binaries built with Visual C++ 2017 are now available from this page on my company's site: 
http://objexx.com/OpenSceneGraph.html

	Release and debug builds are provided.

	osgQt built from the git master head is included.

	The  curl, freetype, GDAL, giflib, glut, jpeg, lpng, minizip, tiff, and zlib plugins are included and the versions are shown on the page.

	The GDAL plugin is new in the OSG 3.6.3 builds.

	A ReadMe.txt file in the root directory of the expanded package has some details. 

	These are 64-bit builds with the default GL2 OpenGL level.

	Other builds and additional plugins may be provided upon request.



Best regards,
