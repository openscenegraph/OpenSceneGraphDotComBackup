On 5 January 2017 at 10:46, Bhanu Chandra <bhanuchandra151 at gmail.com> wrote:
> What is the difference between build with --xx & --yy and build without  --xx & --yy commands using VirtualPlanetBuilder tool to the geospatialised files

The --xx and --yy command line options only exist as a fallback for
data that has no geospatial coordinates such as jpeg.  I would
recommend using VPB with geospatial image files and DEMS as specifying
the coordinates of individual files will be error prone and tedious.
