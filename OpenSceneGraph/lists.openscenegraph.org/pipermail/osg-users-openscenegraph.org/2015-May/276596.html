Hi again,

I repeated my measurements with a pure OpenGL application and compared them to the previously recorded data from OSG:

OpenGL + Nvidia Defaults + Vsync On Min: 56.7 Max: 65.0 Avg: 60.8
OpenGL + Nvidia Defaults + Vsync Off Min: 2.9 Max: 25.0 Avg: 14.6
OSG 3.2.1 + Nvidia Defaults + VSync On Min: 59 Max: 65 Avg: 62
OSG 3.2.1 + Nvidia Defaults + VSync Off Min: 6 Max: 21 Avg: 14 

So it seems that OSG does not add any additional latency compared to a pure OpenGL application.

I repeated the measurements with Windows Desktop Window Manager both on and off, but these measurements had no discernible difference.
