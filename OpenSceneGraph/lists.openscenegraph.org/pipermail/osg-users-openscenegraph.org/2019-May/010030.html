Hi Nikita,

I have a window of opportunity to get 3.6.4 out but didn't get enough
feedback from the community on testing to know whether it was ready
during this window of time I had.  I'm back to working close to 100%
of my time on the VSG side right now.  If you need fixes that are
going into the final 3.6.4 release then the best thing for the now
will be to use the 3.6 branch as this will contain all the
improvements that will make it into 3.6.4.

When I get quieter period of dev I'll go for another release attempt.
I can't make releases on my own though, I need to community to test
across the range of platforms we have to be able to know whether the
code is ready.

Cheers,
Robert.
