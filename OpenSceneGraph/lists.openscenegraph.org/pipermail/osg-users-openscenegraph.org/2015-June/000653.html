> Someone ?
> The file got 10 times bigger! Why would it happen ?
> (675 kb into 6400 kb)


I would have though it obvious - the extra size is to store the images you
have told the writing to inline in the file.  By default the images are
stored in raw memory format without any additional compression that you'd
get with jpeg.  You can get the .osgb writing to inline using the
WriteImageHint=IncludeFile.
